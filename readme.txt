== Description ==
Contrast Content Generator allows you to generate dummy lorem ipsum content in Wordpress. You can generate posts, pages or any custom post types you have created. It is especially useful for testing larger navigations or in situations where you need a lot of content ie. pagination tests. The menu is available in tools -> generate content. 

If you have an interest in translating this plugin please let us know. Current languages are english and finnish.

We do not offer any support for Contrast Content Generator

== Installation ==
1. Install the Contrast Content Generator plugin
2. Activate the plugin
3. Navigate to Tools -> Generate Content
4. Choose content type and amount, generate.
